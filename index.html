  • Photon Tracer

    Ambient occlusion, global illumination and soft shadows

    I am working on a photon tracer, which is basically a pathtracer but slower.

    Instead of tracing from the camera, I'm tracing form the lightsources, because this is more like light works in the real world, I can add ambient occlusion, global illumination and soft shadows with a small amount of code, and it is more accurate than an approximation as well. This photon tracer is written on the CPU, because it is way easier to do it this way, and I am not really planning on actually using it for anything serious.
    I also want to implement stuff like reflection, refraction and physically based materials.

  • Voxel Raytracer

    Ambient occlusion, global illumination and soft shadows

    I worked on a Voxel Raytracer for a school project in GLSL with C#

    Our goal was to create a voxel game, and we thought that raytracing would be fun. Because it needed to be done quickly there isn't really that much to it. It's just a raytracer that can render a voxel mesh.
    We used checkerboard rendering to make sure that it ran at a high enough fps. And we also didn't implement refraction, as we thought it would cause too much of a performance loss and we didn't really need it.
